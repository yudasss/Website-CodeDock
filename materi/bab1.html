BAB 1 - PENDAHULUAN WEB DEVELOPMENT

Memahami arsitektur web, peran browser, client-side vs server-side, dan alat kerja dasar.

Sub-bab 1.1: Arsitektur Web & Client-Server

Web development adalah proses membangun dan memelihara website. Pemahaman dasar dimulai dari bagaimana browser (Client) berkomunikasi dengan Server (Host).

  • Client-Side: Kode yang dieksekusi di browser pengguna (HTML, CSS, JavaScript).
  • Server-Side: Kode yang dieksekusi di server (PHP, Python, Node.js, dll.).
// Contoh Kode Sisi Klien (JavaScript)
document.addEventListener('click', () => {
    console.log('Interaksi ditangani oleh browser.');
});

Sub-bab 1.2: Tools Wajib

Beberapa elemen penting yang harus dimiliki setiap pengembang web:

  • Browser Modern: Chrome, Firefox, Edge (untuk testing).
  • Code Editor: VS Code atau Sublime Text.
  • DevTools: Fitur inspeksi bawaan di browser untuk debugging.
<!-- Pastikan struktur dasar HTML selalu benar -->
<!DOCTYPE html>
<html lang="id">
<head>
    <meta charset="UTF-8">
    <title>Judul Halaman</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
</body>
</html>
