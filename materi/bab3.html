BAB 3 - STYLING MODERN DENGAN TAILWIND CSS

Mempelajari framework CSS utilitas-first untuk membangun antarmuka yang cepat.

Sub-bab 3.1: Konsep Utility-First

Tailwind CSS memungkinkan Anda menata elemen langsung di HTML menggunakan kelas utilitas (*utility classes*). Ini meningkatkan kecepatan pengembangan dan konsistensi desain.

  • Contoh: `bg-blue-500`, `text-white`, `p-4`, `shadow-lg`.
  • Penting: Harus di-build, tetapi untuk demo kita asumsikan sudah tersedia.
<div class="bg-indigo-600 text-white p-4 rounded-lg shadow-xl">
    <h2 class="text-xl font-bold">Ini Styling Tailwind</h2>
</div>

Sub-bab 3.2: Desain Responsif dengan Utilitas

Tailwind menggunakan prefix responsif (*responsive prefixes*) seperti `sm:`, `md:`, dan `lg:` untuk menerapkan gaya spesifik berdasarkan lebar layar.

Ini menyederhanakan media queries yang kompleks di CSS murni.

<div class="p-4 bg-red-400 md:bg-blue-400 lg:bg-green-400">
    <!-- Latar belakang akan berubah: Merah (default), Biru (medium), Hijau (large) -->
</div>
