Tentang Pembuat dan Tujuan Website

Tujuan Website

Website CodeDock ini dibuat sebagai bagian dari pemenuhan Ujian Akhir Semester (UAS) mata kuliah Pemrograman Web 1. Tujuan utamanya adalah untuk mendemonstrasikan kemampuan mahasiswa dalam:

  • Membangun website statis yang fungsional dan terstruktur.
  • Mengaplikasikan konsep Responsive Design menggunakan HTML dan CSS Media Queries.
  • Menyediakan konten edukatif dalam format bab dan sub-bab.
  • Mengimplementasikan interaktivitas sederhana menggunakan JavaScript.
